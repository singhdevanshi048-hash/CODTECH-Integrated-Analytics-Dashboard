# 📊 CODTECH Integrated Analytics Dashboard

## 📌 Project Overview

The **CODTECH Integrated Analytics Dashboard** is a professional data analytics and machine learning dashboard developed using **Python, Dash, Plotly, Pandas, and Scikit-Learn**.

This project integrates **Big Data Analytics**, **Interactive Data Visualization**, and **Machine Learning Prediction** into a single dashboard, providing users with meaningful insights through interactive charts and predictive analytics.

---

# ✨ Features

### 🏠 Home Dashboard

* Interactive KPI Cards
* Total Revenue
* Total Orders
* Total Students
* Average Math Score
* Clean and Professional Dashboard UI

### 📊 Amazon Analytics

* Revenue by Category
* Revenue by Country
* Top 10 Products Analysis
* Country-wise Revenue Analysis

* Interactive Pie Charts

### 🎓 Student Analytics

* Interactive Visualizations

# ⚙ Installation

## Clone the Repository

```bash
git clone https://github.com/singhdevanshi048-hash/CODTECH-Integrated-Analytics-Dashboard.git
```

## Navigate to the Project Folder

```bash
cd CODTECH-Integrated-Analytics-Dashboard
```

## Install Required Libraries

```bash
pip install dash pandas plotly scikit-learn
```

## Run the Dashboard

```bash
python dashboard.py
```

Open your browser and visit:

```text
http://127.0.0.1:8050/
```
