# 📊 CODTECH Integrated Analytics Dashboard

## 📌 Project Overview

The CODTECH Integrated Analytics Dashboard is a professional data analytics and machine learning dashboard developed using Python, Dash, Plotly, Pandas, and Scikit-Learn.

This project combines:

* Amazon Sales Analytics
* Student Performance Analytics
* Machine Learning Prediction
* Interactive Dashboard Visualization

The dashboard provides business insights through interactive charts, KPI cards, and predictive analytics in a single platform.
